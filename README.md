# Квиз «Столицы мира» на Django

Веб-приложение для проверки знаний столиц мира. Реализовано на Django с использованием Bootstrap и собственных CSS-стилей (светло-зелёная тема).

## Katya

Екатерина

## Функционал

- Выбор количества вопросов (5, 10, 15)
- Случайные вопросы из базы данных
- Проверка ответа с обратной связью
- Подсчёт правильных ответов
- Добавление новых вопросов через форму (с валидацией)
- Встроенная админ-панель Django для управления вопросами

## Технологии

- Python 3
- Django 5.x
- SQLite
- Bootstrap 5
- HTML / CSS
